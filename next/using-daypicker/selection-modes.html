Selection Modes

DayPicker comes with some predefined rules for selecting days:

  • Single mode: only one day can be selected.
  • Multiple mode: select multiple days.
  • Range mode: select a continuous range of days.
  • Create custom selection rules for more complex scenarios.

To make days selectable, set the mode prop. Disallow the selection of specific days by using the disabled prop.

Prop NameTypeDescription
mode"single" | "multiple" | "range" | "none"Set a selection mode. Default is "none".
disabledMatcher | Matcher[]Disabled days that cannot be selected.

When the mode prop is set to "single", only one day can be selected at a time.

Single Mode Propsâ€‹

Prop NameTypeDescription
selectedDate | undefinedThe selected date.
onSelectSelectSingleEventHandlerEvent callback when a date is selected.
requiredbooleanMake the selection required.

The following code snippet will render a date picker with a single selected date. When a day is clicked, the selectedDate state is updated.

import { useState } from "react";

import { DayPicker } from "react-day-picker";

export function App() {
  const initiallySelectedDate = new Date();
  const [selectedDate, setSelectedDate] = useState(initiallySelectedDate);
  return (
    <DayPicker
      mode="single"
      selected={selectedDate}
      onSelect={setSelectedDate}
    />
  );
}

Multiple Mode Propsâ€‹

Prop NameTypeDescription
selectedDate[] | undefinedThe selected dates.
onSelectSelectMultipleEventHandlerEvent callback when a date is selected.
minnumberThe minimum dates that can be selected.
maxnumberThe maximum dates that can be selected.

Min and Max Datesâ€‹

Use the min and max props to limit the number of selectable dates.

import { addDays } from "date-fns";
import { DayPicker } from "react-day-picker";

export function MultipleMinMax() {
  const defaultSelected = [new Date(), addDays(new Date(), 1)];
  return (
    <DayPicker
      defaultSelected={defaultSelected}
      mode="multiple"
      min={2}
      max={5}
    />
  );
}

Range Modeâ€‹

When the mode prop is set to "range", DayPicker allows selecting a continuous range of dates.

<DayPicker mode="range" />
Prop NameTypeDescription
selectedDateRangeThe selected range.
onSelectSelectRangeEventHandlerEvent callback when a date is selected.
minnumberThe minimum dates that can be selected.
maxnumberThe maximum dates that can be selected.
import { useState } from "react";

import { addDays } from "date-fns";
import { DateRange, DayPicker } from "react-day-picker";

export function Range() {
  const initialRange: DateRange = {
    from: new Date(),
    to: addDays(new Date(), 4)
  };

  const [range, setRange] = useState<DateRange | undefined>(initialRange);

  return <DayPicker mode="range" selected={range} onSelect={setRange} />;
}

Min and Max Datesâ€‹

Also in range mode, you can set the min and max props to limit the number of selectable dates.

import { useState } from "react";

import { DateRange, DayPicker } from "react-day-picker";

export function RangeMinMax() {
  const [range, setRange] = useState<DateRange | undefined>();

  return (
    <DayPicker
      defaultMonth={new Date(2022, 8)}
      mode="range"
      min={3}
      max={6}
      selected={range}
      onSelect={setRange}
    />
  );
}

Disabling Datesâ€‹

To disable specific days, use the disabled prop. The prop accepts a Matcher or an array of matchers that can be used to make some days not selectable.

<DayPicker
  mode="single"
  disabled={{ daysOfWeek: [0, 6] }} // Disable Sundays and Saturdays
/>
